# 🔥 enjou-sns — 炎上！SNS

Game of Life応用のSNS炎上シミュレーションゲーム。

## 概要
セル・オートマトンを応用して、SNSの炎上拡散と鎮火の動的バランスを再現するシミュレーションゲーム。

- 🔥 **flame** vs 💧 **water** vs 🟢 **brand**
- 👑 インフルエンサー / 🔑 キーノード / ❄️ 凍結 / 📺 古メディア
- 🟢 ハッカー / 🟡 百科事典 などの介入要素
- 難易度: easy / normal / hard / custom
